Эффективное управление проектами
Санкт-Петербург 
28 - 30 апреля 2008 г.
Цель мероприятия: представить систематизированные базовые знания в области управления проектами, рассмотреть способы повышения качества управления проектами, применить техники анализа эффективности проектов

Основные блоки программы:

1. Сущность проектного управления, процессы управления проектами.

2. Процесс планирования как основной элемент управления проектом.

3. Критерии успешного управления проектом.

4. Жизненный цикл проекта и баланс ресурсов при старте нового проекта.

5. Процессы жизненного цикла проекта.

6. Ключевые фигуры проекта - задачи и интересы.

7. Ответственность ключевых фигур за успех всех стадий проекта.

8. Основная и вспомогательная компоненты процесса исполнения и контроля.

9. Анализ плана проекта и процесса исполнения проекта.

10. Процесс управления исполнением проекта, процесс завершения проекта.

11. Сопротивление нововведениям, характеристики эффективной команды управления проектами.

12. Зоны ответственности продавца, руководителя проекта и эксперта по качеству.

13. Задачи контроля качества проекта.

14. Microsoft Project - средство доступное каждому исполнителю проекта.

15. Практическая разработка проекта.
